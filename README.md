# RIG Community Data Engineering Scholarship — Student Portfolio

This repository is a personal portfolio for the student Shon Lei Zaw in the RIG Community Data Engineering Scholarship.

## Purpose
- Keep personal assignments and project work under version control.
- Save lecture notes and resources used for learning.

## Suggested Folder Structure
- `lectures/` — lecture notes, slides, and references (name files like `YYYY-MM-DD - Topic.md`)
- `assignments/` — assignment prompts and your solutions
- `portfolio/` — polished projects, final reports, demo links
- `resources/` — extra readings, links, datasets

## File Naming Convention
For day-based course structure use `Day-1`, `Day-2`, etc. Examples:
- Assignment file: `Day-1_Assignment-1_YourName.md` or `Day-2_Assignment-2_YourName.md`
- Lecture notes: `Day-1_Lecture-Notes.md`
- Submission archive: `YourName_Day-1_Assignment-1_2026-05-25.zip`
